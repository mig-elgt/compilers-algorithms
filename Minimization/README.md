# Minimización de un AFD

El objetivo de este programa es implementar el algoritmo de minización para generar un AFD Mínimo a partir de AFD. Este AFD Mínimo es el Autómata utilizado para llevar acabo el análisis léxico del compilador.

[Consultar Algoritmo](http://web.ing.puc.cl/~jabaier/iic2222/minimizacion.pdf)
