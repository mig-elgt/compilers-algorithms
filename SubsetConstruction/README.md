# Construcción de Subconjuntos

Este programa consiste en construir un AFD a partir de un AFND, la construcción de AFND ya viene incluida en el programa. El objetivo de este algoritmo es obtener un AFD equivalente donde este puede reconocer el mismo lenguaje regular defino por una gramática regular.
