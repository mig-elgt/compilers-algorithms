# Autómatas Finitos No Deterministas con transiciones vacías (AFND-e)

Este programa consiste en implementar el algoritmo de Thompson, para obtener autómatas fínitos no deterministas con transiciones vacías a partir de expresiones regulares. El programa incluye el módulo de gramáticas regulares para generar una expresión regular y aplicar el algoritmo de Thompson.

[Consultar Algoritmo de Thompson](https://es.wikipedia.org/wiki/Algoritmo_de_Thompson)
